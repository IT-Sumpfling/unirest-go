Setup
-------------
To setup your project, follow these steps:

 1. Download Unirest-Go using <code>go get github.com/apimatic/unirest-go</code>.
 2. Import Unirest-Go in your project using

 ```go
 import "github.com/apimatic/unirest-go"
```
